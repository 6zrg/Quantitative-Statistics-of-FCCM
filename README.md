# Quantitative-Statistics-of-FCCM
## 1. Environment

- Please prepare an environment with python=3.8, and then use the command "pip install -r requirements.txt" for the dependencies.

## 2. Train/Test

- Train
```bash
python train.py
```

- Test
```bash
python infer.py
```

```bash
python focus_statistics_3D.py
```
